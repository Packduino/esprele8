<h2>EspRele8</h2>
Placa para automação residencial de baixo custo com ESP01 e CHIP Arduino UNO
 <br>- Possui 8 entradas
 <br>- Possui 8 saídas
 <br>- Acesso via Wifi
<br> - Testando com MQQT
